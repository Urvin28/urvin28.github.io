Diabetes Classification

The original script has been added to this folder. To run locally:

  1. Place the dataset CSV `diabetes_dataset.csv` in this folder (same directory as `diabetes_classification.py`).
  2. Install dependencies (see `requirements.txt` in `projects/ml/`).
  3. Run: python diabetes_classification.py

Files in this folder:
